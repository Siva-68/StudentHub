# StudentHub

A Full Stack Student Management System built using MERN Stack.

## Features

- Admin Authentication (JWT)
- Student CRUD Operations
- Search & Filter Students
- Pagination
- Dashboard Analytics
- Responsive UI
- Protected Routes
- REST API Architecture

## Tech Stack

### Frontend
- React
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT
- Bcrypt

## Project Structure

/client → React Frontend

/server → Express Backend

## Installation

### Clone Repository

git clone <repo-url>

### Frontend

cd client
npm install
npm run dev

### Backend

cd server
npm install
npm run dev

## Environment Variables

Create .env file using .env.example
